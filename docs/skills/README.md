# Shared Skills

This repo vendors shareable Codex skills under `docs/skills/`.

## Orchestrate

The `orchestrate` skill is mirrored from the local Codex skill home so it can be reviewed, versioned, and installed from this repo.

Repo path:

- `docs/skills/orchestrate/`

Install locally:

```bash
./docs/install-orchestrate-skill.sh
```

That copies the vendored skill into:

- `${CODEX_HOME:-$HOME/.codex}/skills/orchestrate`
